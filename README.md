# Sticky Notes
Note android app built with Kotlin, using Clean Architecture, with Jetpack Compose.
<br>
## Features
- Add notes
- Edit notes
- Delete notes
- Change note color
- Sort notes by [Date - Title - Color]

## Built with
- Kotlin
- Clean Architecture
- ViewModel
- Jetpack Compose
- Kotlin Coroutines
- Navigation
- Dagger-Hilt
- Room
